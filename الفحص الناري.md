## نطاق الفحص المنفذ قبل الإيقاف

- فحص صحة الخدمة محليًا على `http://localhost:5174`.
- فحص تحمل أولي باستخدام `autocannon` على:
  - `/api/health/live`
  - `/api/market/overview`
- فحص baseline للبناء:
  - `npm run build` ناجح.
  - `npm run lint` يحتوي أخطاء موجودة مسبقًا خارج نطاق تغييرات التحمل.
